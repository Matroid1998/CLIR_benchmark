# attic

Frozen records. Nothing in this directory is maintained, imported, tested, or
expected to run.

These are one-off and superseded scripts carried over verbatim from the old
repo, [`Multi-Lingual-QAC`](../../Multi-Lingual-QAC). Each file keeps its
original bytes with a header comment prepended saying what it did, when it ran,
why it is retired, and what replaced it. Their imports and paths still point at
the old repo's layout and were deliberately not fixed — repointing them would
make them look runnable, which they are not.

They are here because several of them produced artifacts the project still
depends on (the regraded QAC files, the human-agreement numbers in the paper,
the published Alias-Graph dataset), and because a few document decisions that
would otherwise be invisible — most of all why source documents are never
machine-translated. Read the old repo for surrounding context.
